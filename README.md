# Industrial Automation Projects

A hands-on portfolio spanning TwinCAT 3, Allen-Bradley Studio 5000, and Factory I/O covering PLC programming in IEC 61131-3 Structured Text and Ladder Logic, PackML state machines, and OOP function block design built as a practical supplement to an industrial automation career transition.

## Projects

| Project | Platform | Topics | Status |
|---------|----------|--------|--------|
| Bottle Filling Simulator | TwinCAT 3 | PackML, Structured Text, State Machines | In Progress |

## Platforms

- **TwinCAT 3** — Beckhoff's PC-based automation platform supporting IEC 61131-3 languages and object-oriented extensions in a Visual Studio-integrated environment.
- **Allen-Bradley Studio 5000** — Rockwell Automation's Logix Designer environment for programming ControlLogix and CompactLogix PLCs using Ladder Logic, Structured Text, and Function Block Diagram.
- **Factory I/O** — A 3D industrial simulation tool that connects to real PLC runtimes, enabling hardware-free testing of automation programs against realistic virtual processes.
